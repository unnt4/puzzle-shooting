# PuzzleShooting（パズルシューティング）

## ■ 概要
パズルを揃えて自機を強化し、敵を倒す「パズル × シューティング」ゲームです。
画面下部のパズルエリアでドロップを操作し、コンボを繋げることでショットが強化されます。


## ■ 動作環境
- OS: Windows 11
- 言語: Python 3.11
- ライブラリ: pyxel,PyxelUniversalFont(日本語表示用追加ライブラリ）

## ■ ゲームの遊び方
### 1. 基本ルール
- 自機を操作して敵の弾幕を避けながら、パズルを揃えて自機を強化します。
- パズルのコンボ数に応じて、自機のショットが強化されます。
- 敵の弾に当たるとLIFEが減り、0になるとゲームオーバーです。

### 2. 操作方法

| キー | 動作 | 詳細 |
|:---:|:---|:---|
| **↑↓→←** | 自機の移動 | 矢印キーで自機を上下左右に操作します。 |
| **Enter** | ステージ決定 | Enterキーで選択したステージを開始します |
| **Space** | パズル操作 | **長押し**でドロップを掴み、**移動**させて離すと消えます（パズドラ風操作）。 |
| **X** | ボム発動 | 画面上の全ての敵弾を消滅させます（緊急回避用）。 |
| **P** | ポーズ | ゲームを一時停止します。 |
| **S** | ステージ選択 | ステージ選択画面に戻ります。 |
| **R** | リトライ | ゲームオーバー時などに最初からやり直します。 |
| **M** | デバッグモード | 自機が無敵状態になります（動作確認用）。 |
